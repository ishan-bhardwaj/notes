# JavaScript

- High-level, object-oriented, multi-paradigm programming langugage.
- Inline JS (inside `<head>`) -
```
<script>
    // Add JS code
</script>
```

- Linking `script.js` file (generally added before the end of `<body>`) -
```
<script src='script.js'></script>
```

> [!WARNING]
> Self closing (`<script src='script.js' />`) doesn't work with script tags.
