# C++

- Compiler version - `g++ --version`
- Hello World -
  ```
  #include <iostream>

  int main() {
    std::cout << "Hello World";
    return 0;
  }
  ```

- Reading prompt -
  ```
  int num;
  std::cout << "Enter number: ";
  std::cin >> num;
  std::cout << "You entered: " << num << std::endl;
  ```
