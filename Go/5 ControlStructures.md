# Control Structures

## If-else

- Syntax -
```
if <condition> {
    ...
} else if <condtion> {
    ...
} else {
    ...
}
```

## switch

- Syntax -
```
a := 10

switch a {
    case 1: ...
    case 2: ...
    default: ...
}
```

## for

- Syntax -
```
for i := 0; i < 10; i++ {
    <...>
}
```

> [!TIP]
> Use `break` & `continue` to break out or continue the loop.
